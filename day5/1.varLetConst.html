변수 선언할때 무엇을 썼냐? [ var ] 위에거는 옛날 구닥다리 방식이고 지금 쓰는건 [ let, const ] 왜 바꿨는가? -> var에 문제점이 많아서 바꿨다. 어떤 문제점들이 있었을까? 예를들어 는 이런식으로 지멋대로 읽어버린다. 이렇게 var로 선언된 변수를 미리 읽어서 갖고있는게 [ 호이스팅 ] 이라는 개념이다. 변수 중복 선언도 호이스팅 되면 이렇게 되기때문에 var를 대체할만한게 필요했다. 그래서 이용하는게 let : 변수선언 const : 상수선언 ex) let 현재주식가격; const 생년월일; 이 프로그램이 실행 되는동안에 바뀌지 않는건 const
